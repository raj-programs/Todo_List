``` Interactive Alarm Clock ```

A simple **interactive alarm clock** built using **Python** and **Tkinter**. This project allows you to set alarms, snooze, and stop them, all from a graphical user interface (GUI).

---

## Features

- Set an alarm for a specific time (hours, minutes, seconds)
- Snooze the alarm for 5 minutes
- Stop the alarm
- GUI interface using Tkinter
- Plays a sound file (`alarm.mp3`) when the alarm triggers
