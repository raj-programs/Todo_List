``` Personal Portfolio Webpage ```

## Overview
This is a **personal portfolio webpage** designed to showcase skills, projects, experience, and contact information in a clean, modern, and tech-themed layout. The webpage is built using **HTML, CSS**, and incorporates **Font Awesome icons** for social media links.

The design follows a **dark, futuristic tech theme** with accent colors to highlight projects, skills, and interactive elements.

---

## Features

### Sections
1. **Header**
   - Displays name and a brief introduction.
   - Navigation links for smooth scrolling to different sections.

2. **About Me**
   - Detailed bio explaining background and interests.
   - Lists qualifications and achievements.

3. **Projects**
   - Highlights personal projects with images, descriptions, and technologies used.
   - Projects include:
     - Expense Tracker (Python CLI)
     - Rock-Paper-Scissors Game (C++ Console)
     - Todo List App (HTML, CSS, JavaScript)

4. **Skills / Expertise**
   - Cards highlighting key skills like C++, HTML & CSS, JavaScript.
   - Each card includes a description and related tech stack.

5. **Experience**
   - Timeline layout showing work experience and internships.
   - Details include position, company, duration, responsibilities, and tech stack.

6. **Contact**
   - Email and social media links using **Font Awesome icons**.
   - Styled buttons for easy accessibility.

7. **Footer**
   - Displays copyright information.

---

## Technologies Used
- **HTML5**: Structure of the webpage.
- **CSS3**: Styling, layout, hover effects, card designs, responsive design.
- **Font Awesome**: Social media icons.

---

## Design & Theme
- **Dark Tech Theme**:
  - Body background: `#0D1117`
  - Section backgrounds: `#1A1F29`
- **Accent Colors**:
  - Neon Green: `#00FF7F` (highlights, keywords)
  - Electric Purple: `#9B59B6` (headings/subheadings)
  - Tech Blue: `#00AFFF` (optional accents)
- **Cards**:
  - Slight shadow and hover effect with neon glow.
  - Rounded corners and consistent spacing.
- **Responsive Design**:
  - Flexible layouts for projects and skills sections.
  - Adjustments for smaller screen sizes.
